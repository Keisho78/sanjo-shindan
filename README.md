# 三条工務店 ビジネス版MBTI適性診断

採用選考の場で応募者に受けてもらう参考ツールです。MBTIタイプの選択とビジネス特化質問76問への回答から、カルチャーフィット度と6部門（営業・工務・空間デザイナー・財務経理/契約管理・経営企画/人事/広報・雅家）への適性傾向を表示します。

**本診断は合否や配属を決定するものではなく、応募者の自己理解と面接での対話のきっかけづくりを目的とした参考情報です。**

## 使い方

- `index.html` をブラウザで開くだけで動作します（サーバー不要）
- 回答はブラウザのメモリ上でのみ処理され、外部への送信・保存は一切ありません

## 技術構成

- 単一の `index.html`（React 18 + Babel + Chart.js をCDN読み込み）
- 質問・スコア重み・タイプ補正のカスタマイズ方法は `index.html` 冒頭のコメントを参照
