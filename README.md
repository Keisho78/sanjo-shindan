# 三条工務店 ビジネス版MBTI適性診断

採用選考の場で応募者に受けてもらう参考ツールです。MBTIタイプの選択とビジネス特化質問76問への回答から、カルチャーフィット度と7部門（営業・工務・空間デザイナー・財務経理・契約管理・経営企画・人事広報）への適性傾向を表示します。結果には16種類の動物イラストキャラクター（例：ゴリラ現場監督、クロネコ参謀）が表示されます。

**本診断は合否や配属を決定するものではなく、応募者の自己理解と面接での対話のきっかけづくりを目的とした参考情報です。**

## 使い方

- `index.html` をブラウザで開くだけで動作します（サーバー不要）
- 回答はブラウザ内でのみ処理され、外部への送信は一切ありません
  （リロード事故対策として回答中のみ sessionStorage に一時保存し、結果表示の時点で自動削除します）

## 主な機能

- MBTIタイプ選択（4軸 or 16タイプ一覧）＋ ビジネス特化質問76問
- カルチャーフィット度（円形ゲージ）・部署別適性（レーダーチャート）・特性の詳細分析
- 回答途中でリロードしてもトップ画面の「続きから再開する」で復帰可能
- キーボード操作対応（1〜5 で回答、←→ で質問移動）
- 結果画面の「結果を印刷 / PDF保存」ボタンで印刷用レイアウトを出力

## デザイン

- 上質ミニマルな和モダン（クリーム×ベージュ×ダークブラウン・明朝の細い見出し・広い字間・ヘアライン罫線）
- キャラクターはゆるいイラスト調SVGを円形メダリオンで上品に表示
- 見出し: Shippori Mincho / 英字・数字: Jost / 本文: Noto Sans JP
- デザイン違い: `design/retro`（ファミコン風）・`design/yamakachi`（手描きイラスト調）ブランチあり

## 技術構成

- 単一の `index.html`（React 18 + Babel + Chart.js + Google Fonts をCDN読み込み）
- 質問・スコア重み・タイプ補正のカスタマイズ方法は `index.html` 冒頭のコメントを参照
